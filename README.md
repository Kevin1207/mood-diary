# 心情日记 🌈

一个简洁美观的心情记录网页应用，帮助你记录每一天的心情，回顾美好时光。

## 功能特点

✨ **主要功能**
- 📅 选择日期记录心情
- 😊 6种心情选项（非常开心、开心、平静、疲惫、难过、生气）
- 📝 添加心情笔记（可选）
- 📆 月历视图查看心情记录
- 💾 本地存储，数据持久化
- 🗑️ 删除不需要的记录

🎨 **设计亮点**
- 现代化渐变背景
- 流畅的动画效果
- 响应式设计，支持手机和电脑
- 直观的用户界面
- 优雅的弹窗展示

## 使用方法

### 1. 打开应用
直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 记录心情
1. 选择日期（默认为今天）
2. 点击对应的心情图标
3. （可选）输入心情笔记
4. 点击"保存心情"按钮

### 3. 查看历史记录
- 使用左右箭头切换月份
- 日历中有心情图标的日期表示已有记录
- 点击任意日期查看详细记录
- 未记录的日期点击后会跳转到记录页面

### 4. 管理记录
- 在详情弹窗中可以删除记录
- 数据自动保存在浏览器本地存储中

## 数据存储

所有心情数据都保存在浏览器的 LocalStorage 中，格式如下：

```json
{
  "2026-02-10": {
    "mood": "happy",
    "note": "今天天气真好！",
    "timestamp": "2026-02-10T09:26:00.000Z"
  }
}
```

⚠️ **注意**：清除浏览器缓存会删除所有数据，请谨慎操作。
